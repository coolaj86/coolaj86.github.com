--- layout: article name: gumstix-overo-serial-usb-with-os-x-106 title: Gumstix Overo Serial-USB with OS X 10.6 Snow Leopard categories: thesystemisntdown uncategorized time: 2010-07-23 13:39:00 -06:00 --- In a previous post I was using the PL2303 adapter. Now I'm using an FTDI that's built onto the Tobi board.

  • Download and install the driver from FTDI
  • reboot
  • ls /dev/tty*
  • /dev/tty.usbserial*
  • /dev/tty.usbserial-A100e0TP (this may change when unplugging and replugging)
  • sudo minicom -s (setup the port as the tty.usbserial-xxxxx)
Works for me!


This is reported to also work for OSX 10.3 Panther, 10.4 Tiger, 10.5 Leopard
